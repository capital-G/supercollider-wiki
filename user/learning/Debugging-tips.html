Debugging tips¶

QtGUI Debug Capability¶

Purpose¶

Print verbose debug messages while diagnosing issues with the Qt GUI features.

Usage¶

Only enabled on a debug build.

QtGUI.debugLevel_(0) // 0 = no output, 3 = verbose
